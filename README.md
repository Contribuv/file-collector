## 📖 简介

文件收集器是一个功能完备的轻量级 Web 应用，支持**多用户体系**、**Office 在线预览**和**双模式分享**。管理员可在后台创建多个独立的收集链接，每个链接拥有独立通行证、上传限制和有效期。每个链接自动生成**收集页**（上传）和**分享页**（浏览下载）两种模式。内置邀请码注册系统，支持管理员和普通用户两种角色。Office 文件（Word/Excel/PPT）可直接在线预览，无需下载。

**适用场景：** 团队文件汇总、作业收集、资料分发、协作共享

**适用平台：** [飞牛 fnOS](https://www.fnnas.com/) Native 应用（.fpk 格式），支持 **x86 / ARM / LoongArch / RISC-V** 全平台。

| 功能 | 说明 |

| 📄 **Office 预览** | Word / Excel / PPT 在线预览（JIT Viewer SDK），无需下载 |

| 🎬 **多媒体预览** | 图片 / PDF / 视频 / 文本文件在线预览 |
| 📏 **灵活限制** | 自定义单文件大小上限（GB）和上传数量上限（0 = 不限制） |

| 🎫 **邀请码系统** | 管理员生成邀请码（可设有效期），用户凭码注册 |
| 📦 **数据备份** | 一键下载 / 导入 SQLite 数据库，自动验证和备份旧库 |
| 🔒 **安全防护** | CSRF 保护、登录频率限制、通行证验证速率限制、会话安全、危险文件拦截 |
| 📱 **响应式 UI** | 桌面和移动端完美适配，微信浏览器自动提示跳转 |
| 🎨 **深度定制** | 自定义站点标题、登录提示、页尾文字、首页开关、公网网址 |

### 管理员后台

- **仪表盘** — 总览统计（链接数 / 记录数 / 存储用量 / 活跃链接）+ 最近上传
- **收集链接** — 创建、编辑、启停、删除、一键复制收集 / 分享链接
- **上传记录** — 分页浏览、按链接过滤、批量删除、在线预览、下载文件
- **用户管理** — 查看用户列表、编辑用户名 / 密码、启用 / 停用 / 删除 / 重置密码
- **邀请码管理** — 生成邀请码（支持设置有效期）、查看使用状态
- **系统设置** — 账号密码修改、默认上传参数、通行证有效期、公网网址、注册开关、禁止文件类型、数据库备份恢复
- **系统信息** — 实时显示数据库路径、上传目录、监听端口、版本更新检测

## 🚀 快速开始

### fnOS 安装

1. 下载 `.fpk` 安装包
2. 在 fnOS 应用管理中点击「手动安装」，选择 `.fpk` 文件
3. 安装向导中填写**上传文件存储目录**（绝对路径，如 `/vol1/收集文件`）
4. 安装完成后访问 `http://[NAS_IP]:5557/admin`

> ⚠️ 此应用强制使用 IP+5557 访问，请通过 Lucky 反向代理实现域名访问。

### 默认账户

| 用户名 | 密码 |
|:---|:---|
| `admin` | `admin123` |

> ⚠️ 首次登录后请立即修改密码！

## 📋 使用说明

### 管理员操作流程

#### 1. 创建收集链接

登录后台 → 点击「收集链接」→「创建链接」：

| 参数 | 说明 |
|:---|:---|
| 收集名称 | 用于标识该链接，收集者和分享页可见 |
| 通行证 | 访问收集页 / 分享页所需的密码（可留空） |
| 描述 | 富文本描述，支持格式化内容和颜色 |
| 最大文件数 | 允许上传的文件数量（0 = 不限制） |
| 单文件上限 | 单个文件最大体积（GB），上限 64GB |
| 有效期 | N 天后 / 指定日期 / 永不过期 |

创建成功后，自动生成两个地址：
- **收集页**：`/collect/<短ID>` — 可上传文件
- **分享页**：`/share/<短ID>` — 仅浏览下载

点击「复制」按钮即可将链接和通行证一并复制，发送给需要的人。

#### 2. 管理上传记录

后台「上传记录」页面可查看所有提交的文件：
- 支持分页、按链接过滤
- 支持批量选择删除
- 点击文件名可在线预览（图片 / PDF / Office / 视频 / 文本）
- 点击下载按钮保存到本地

#### 3. 用户管理

后台「用户管理」页面：
- 查看所有注册用户及其角色
- 编辑用户信息、重置密码
- 启用 / 停用用户账号
- 删除用户（将一并清除其创建的链接）

#### 4. 邀请码

后台「邀请码管理」→「生成邀请码」：
- 可设置有效期（到期自动失效）
- 复制邀请码发送给需要注册的人
- 注册开关可在「系统设置」中一键开启 / 关闭

#### 5. 系统设置

- **修改密码**：修改管理员登录密码（要求 8 位以上，含字母 + 数字）
- **默认上传参数**：设置新链接的默认文件数量和大小上限
- **通行证有效期**：设置通行证缓存时长（分钟）
- **公网网址**：填写反向代理后的域名地址，复制链接时自动使用
- **注册开关**：一键开放或关闭新用户注册
- **禁止文件类型**：自定义禁止上传的扩展名（留空使用默认危险类型拦截）
- **首页介绍页**：开关首页 landing 页面
- **数据库备份恢复**：一键备份下载 / 导入恢复

### 收集者（上传方）操作

1. 管理员发送收集链接（如 `http://IP:5557/collect/abc12345`）和通行证
2. 打开链接，输入通行证（有效期内在同一浏览器无需重复输入）
3. 拖拽或点击上传文件（实时显示进度）
4. 可查看自己的上传历史、预览和下载已提交文件
5. 如管理员允许，可删除自己的上传记录

### 分享页（浏览方）操作

1. 管理员发送分享链接（如 `http://IP:5557/share/abc12345`）和通行证
2. 打开链接，输入通行证
3. 浏览文件列表，支持在线预览（图片 / PDF / Office / 视频 / 文本）
4. 点击下载按钮保存到本地
5. 分享页不含上传功能，只读访问

### 微信用户提示

在微信中打开链接时，页面会自动弹窗提示「在浏览器中打开」，避免微信内置浏览器限制。

## 📤📥 收集与分享（双模式）

每个链接自动生成两个独立页面：

| 模式 | 地址 | 用途 | 功能 |
|:---|:---|:---|:---|
| 📤 **收集页** | `/collect/<link_id>` | 文件上传 | 拖拽上传、批量提交、进度显示、历史记录 |
| 📥 **分享页** | `/share/<link_id>` | 文件分发 | 浏览文件列表、下载文件、在线预览 |

- 两个页面使用**同一通行证**，验证后在有效期内免重复输入
- 分享页不含上传功能，适合将已收集文件开放给团队查阅下载
- 管理员后台一键复制收集链接和分享链接
- 微信浏览器访问时自动提示「在浏览器中打开」

## ⚙️ 环境变量

| 变量 | 用途 | 默认值 |
|:---|:---|:---|
| `DATA_DIR` | 数据库存储目录 | `$TRIM_DATA_SHARE_PATHS/data` |
| `UPLOAD_BASE` | 上传文件存储根目录 | `$TRIM_DATA_SHARE_PATHS/uploads` |
| `PORT` | 监听端口 | `5557` |
| `FLASK_DEBUG` | 调试模式 | `0` |

> 数据库使用 `TRIM_DATA_SHARE_PATHS`（飞牛官方应用文件目录），更新 / 重装不会丢失数据。

## 🔌 API

### 健康检查

```bash
GET /api/status
```

响应：
```json
{
  "status": "running",
  "db_ok": true,
  "upload_dir_exists": true
}
```

### 收集页

| 方法 | 路径 | 说明 |
|:---|:---|:---|
| `GET` | `/collect/<link_id>` | 收集页面 |
| `POST` | `/collect/<link_id>/verify` | 验证通行证 |
| `POST` | `/collect/<link_id>/logout` | 退出通行证 |
| `POST` | `/collect/<link_id>/upload` | 上传文件（multipart） |
| `GET` | `/collect/<link_id>/records` | 获取上传历史（JSON） |
| `GET` | `/collect/<link_id>/download/<record_id>` | 下载文件 |
| `POST` | `/collect/<link_id>/delete_record/<record_id>` | 删除记录 |

### 分享页

| 方法 | 路径 | 说明 |
|:---|:---|:---|
| `GET` | `/share/<link_id>` | 分享页面 |
| `POST` | `/share/<link_id>/verify` | 验证通行证 |
| `POST` | `/share/<link_id>/logout` | 退出通行证 |
| `GET` | `/share/<link_id>/records` | 获取文件列表（JSON） |
| `GET` | `/share/<link_id>/download/<record_id>` | 下载文件 |
