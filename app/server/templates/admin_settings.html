反向代理设置方法

💡 推荐使用项目内置反代，自动获取飞牛证书，分享链接自动使用域名，无需手动填写公网网址。

方式一（推荐）：项目内置反向代理

无需安装额外应用，自动获取飞牛系统证书进行 HTTPS 反代。

  1. 在本页面下方「反向代理」卡片中点击「开始配置」
  2. 选择已配置证书的 DDNS 域名，设置外网端口
  3. 点击「开始反代」，一键启动
  4. 分享/收集链接将自动使用 https://域名:端口，无需手动填写公网网址

方式二：反向代理服务器（laok 开发）

在飞牛应用中心下载 反向代理服务器，由 laok 大佬开发。
如果反代出现问题，请到飞牛社区找 laok 大佬反馈。

打开反向代理服务器，进入「来源设置」页面：

  1. 点击 「添加规则」
  2. 来源设置：
    • 协议选择 HTTPS
    • 主机名填写 127.0.0.1
    • 端口填写外网端口（如 7786）
    • 域名留空，不填
  3. 目标设置：
    • 主机名/IP 填写文件收集器内网地址和端口，如 127.0.0.1:5557
  4. 高级设置：启用 HSTS、强制跳转 HTTPS、保留原始主机头，全部打钩

方式二：Lucky 反向代理

Lucky 支持反向代理、DDNS、SSL 证书自动续签。下载 Lucky 安装后：

  1. 打开 Lucky 后台（默认 16601），点击左侧 「Web 服务」
  2. 点击 「添加 Web 服务规则」
  3. 填写规则配置：
    • 监听端口：填写外网访问端口，如 7786
    • 启用 TLS：开启，选择飞牛证书或 Lucky 自动申请
    • 前端域名：填写 DDNS 域名，如 nas.fons.com
  4. 切换至「后端」标签：
    • 后端地址：http://127.0.0.1:5557
    • 后端类型：选择 proxy
  5. 点击 「添加」 保存，等待生效
